🚀 PRÓXIMOS PASSOS:

1. INSTALE POSTGRESQL
   - Windows: https://www.postgresql.org/download/windows/
   - Mac: brew install postgresql
   - Linux: sudo apt install postgresql

2. CRIE O BANCO DE DADOS
   Abra SQL Shell ou pgAdmin e execute:
   
   CREATE DATABASE reserva_salas;
   CREATE USER usuario WITH PASSWORD 'senha';
   GRANT ALL PRIVILEGES ON DATABASE reserva_salas TO usuario;

3. CONFIGURE O .env
   - Copie .env.example para .env
   - Edite com suas credenciais PostgreSQL

4. INICIE A APLICAÇÃO
   
   Windows:  run.bat
   Linux:    ./run.sh
   Mac:      ./run.sh

   Ou manual:
   python -m venv venv
   (ativar ambiente virtual)
   pip install -r requirements.txt
   python app.py

5. ACESSE NO NAVEGADOR
   
   Página Principal:    http://localhost:5000
   Painel Admin:        http://localhost:5000/admin

📊 BANCO DE DADOS:

As salas são cadastradas no banco de dados e podem ser gerenciadas pelo painel administrativo.

Você pode adicionar novas salas no painel admin ou diretamente no banco de dados.

🔌 API REST:

GET    /api/salas                    - Listar salas
POST   /api/salas                    - Criar sala
DELETE /api/salas/{id}               - Deletar sala

GET    /api/reservas                 - Listar reservas
POST   /api/reservas                 - Criar reserva
DELETE /api/reservas/{id}            - Cancelar reserva

GET    /api/disponibilidade          - Horários livres
